# EliteTierTag
Centro de descarga del mod de TierTag de EliteStorm

## Versiones disponibles

- **Beta 1.1.0** — No disponible
- **Beta 1.1.1** — Disponible

## Cómo instalar

### 1. Elige tu versión de Minecraft

Selecciona la carpeta de la versión que necesitas. Las versiones disponibles son:
- `mc1.21.1`
- `mc1.21.4`
- `mc1.21.8`
- `mc1.21.11`
- `mc26.1.2`

### 2. Opciones de instalación

En cada carpeta de versión encontrarás dos archivos:

#### Opción A: Usar Modrinth
- Descarga el archivo `.mrpack` (ejemplo: `elite-tiertagger-1.1.1-beta+mc1.21.1.mrpack`)
- Abre Modrinth Launcher
- Importa el archivo `.mrpack` para crear una instancia lista para jugar

#### Opción B: Instalación manual
- Descarga el archivo `.zip` (ejemplo: `elite-tiertagger-1.1.1-beta+mc1.21.1.zip`)
- Extrae el contenido
- Copia los mods a tu carpeta de mods de Minecraft

### 3. ¡Listo para jugar!

Una vez instalado, abre tu launcher de Minecraft y selecciona la versión correcta para comenzar a jugar con EliteTierTag.
